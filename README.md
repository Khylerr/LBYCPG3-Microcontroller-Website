# Silly Core: Microcontroller Hub

A website showcasing information about various microcontroller manufacturers.

## How to Run

Open `index.html` in your browser.

## Features

- Information on major microcontroller manufacturers (STMicro, Microchip, Espressif, Texas Instruments, NXP, Infineon)
- About Us page with developer information
- Disclosure page for AI and image source transparency
